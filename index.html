Voter Fraud Detector

This is a program to detect voter fraud. It works by scanning the official data (see https://data.pa.gov/widgets/mcba-yywm) for any voters over 100.

Usage:

  1. Go to https://data.pa.gov/widgets/mcba-yywm (or any other government vote record) and click Menu > Download > Download As CSV.
  2. Wait for the download to finish.
  3. Press the "Choose File (CSV Only)" button and attach the voting records.
This program is open source. See the source on github to confirm I'm not just making stuff up.
